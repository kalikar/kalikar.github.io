###

一个关于js数组问题。比如：js数组中有 [2,3,4,8,2,2,1,9,3],目的是删除数组中的2。
<pre><code>var arr = [2,3,4,8,2,2,1,9,3];
for (var i = 0; i < arr.length; i++) {
	if (arr[i] == 2) {
		arr.splice(i, 1);
	}
}
console.log(arr);
</code></pre>
得出的结果是[3,4,8,2,1,9,3]，没有达到预期的结果。

原因在于js数组是动态的，应该在删除后i--，这样就可以达到预期的结果。代码如下
<pre><code>var arr = [2,3,4,8,2,2,1,9,3];
for (var i = 0; i < arr.length; i++) {
	if (arr[i] == 2) {
		arr.splice(i, 1);
		i--;
	}
}
console.log(arr);
</code></pre>
